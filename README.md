## 💖 Welcome to Your New Digital Brain
This isn't just any planner—it's a beautifully designed, highly customized dashboard tailored perfectly for you. Featuring a gorgeously vibrant **Petal Rouge** and **Soft Blush** aesthetic (`#fa3869`), this application integrates all aspects of your life into one centralized hub. 

No more scattered notes, missed assignments, or forgotten study sessions. Everything is here, waiting for you! 🌷

---

## ✨ Spectacular Features Built Just for You

### 🎨 Gorgeous Aesthetic & UX
*   **Soft Blush Theme:** A custom UI designed with beautiful pinks, smooth gradients, and glassmorphism elements.
*   **Dynamic Animations:** Engaging bouncy micro-animations powered by Framer Motion to make checking off tasks feel incredibly rewarding!
*   **Fully Responsive:** A beautiful mobile layout so you can take your schedule anywhere—complete with an animated slide-out hamburger menu.

### 📅 College Timetable & Academics
*   **Pre-configured Schedule:** Your exact weekly classes and labs (SP, SPL, IOT, AI/ML Labs in rooms 2305, 2208A, etc.) are deeply integrated into the database. 
*   **Study Blocks:** Automated time-blocking that respects your college hours and creates study slots around them.

### ⏳ The Ultimate Task Manager
*   **Deadline Tracking:** Never forget an assignment again! Set specific date/time deadlines for tasks.
*   **Countdown Badges:** Pending tasks automatically display beautiful red countdowns so you always know exactly how much time you have left. 
*   **Satisfying Completions:** Striking off a task gives instant visual feedback. 

### 📚 UPSC Mission 2027 Tracker
*   **Optional Subject Configured:** Pre-loaded with **Public Administration** as your optional subject.
*   **Macro Roadmap:** A long-term phase-by-phase timeline broken down from now until the 2027 Prelims and Mains.
*   **Progress Rings:** Visual circles showing exactly how many micro-topics you've mastered.

### 🩰 Extra Activities Hub
*   Because life is more than just studying! Manage your personal hobbies, extracurriculars, side hustles, or event planning in a dedicated pipeline.

## 🚀 How to Run Locally

If you ever need to run your command center on your own machine:

### 1. Start the Backend API
```bash
cd backend
npm install
# Make sure your .env has the DATABASE_URL and JWT_SECRET!
npm run start
```

### 2. Start the Beautiful Frontend
```bash
cd frontend
npm install
npm run dev
```
Open `http://localhost:5173` in your browser and prepare to be amazed! 🪄
